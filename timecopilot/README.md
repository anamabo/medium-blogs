# Examples of TimeCopilot

TimeCopilot is a Python library that allows you to do forecasting combining the power 
of large language models and different time series algorithms including the latest foundation models.
In its agent mode, it automates and explains complex forecasting workflows, 
making time series analysis accessible while maintaining professional accuracy.

In the Scripts folder you will find examples of usage of the agent and the forecaster.
For more information about TimeCopilot, visit its GitHub repository [here](https://github.com/TimeCopilot/timecopilot).
