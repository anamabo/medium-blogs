### Requirements

### Set up of the Python environment [MAC]
[The authors of skore recommend to use venv](https://pypi.org/project/skore/), therefore follow the instructions below.

* Make sure you are in the folder, `skore`
* Create a venv with Python>-3.10: 

```/usr/local/bin/Python3.12 -m venv .skore-exploration ```

* Activate your environment:

```source .skore-exploration/bin/activate```

* Install the packages listed in the requirements.txt:

```
pip install --upgrade pip
pip install -r requirements.txt
```
Set up your Python interpreter in your IDE to use the virtual environment you just created.
