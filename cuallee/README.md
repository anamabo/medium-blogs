# Examples of Cuallee

Cualle is a Python library that can be used to get data quality checks. 

In the Notebook you will find 2 examples of usage for Pandas and DuckDB.
For more information about Cuallee, visit its GitHub repository [here](https://github.com/canimus/cuallee).

You can find more information in the blog post [Meet Cuallee: A powerful Python library to automate data quality checks](https://medium.com/data-science-collective/meet-cuallee-a-powerful-python-library-to-automate-data-quality-checks-4aeec6a8d6d1).
